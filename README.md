# ppt-making-sop：PPT制作SOP技能包

这是一个关于 PPT 标准作业程序（SOP）制作的技能文件包，可直接用于 AI 智能体（Agent）配置，或供个人参考使用。

## 📁 文件说明

本仓库包含以下核心文件（位于 `SKILL` 文件夹下）：

- **`ppt-making-sop.md`**：此技能的核心 SOP ，详细介绍了 PPT 制作的步骤、标准和注意事项。**（您可以直接在 GitHub 网页上点击预览此文件）**
- **`ppt-making-sop.zip`**：包含上述 `.md` 文件及其可能相关的配套资源（如有）的压缩包。**（方便您一键下载，用于导入本地 Agent 平台或备份）**

## 🚀 如何使用

1. **直接预览**：点击进入仓库列表中的 `ppt-making-sop.md` 文件，即可直接在 GitHub 网页上阅读教程。
2. **下载使用**：如果您需要把这份技能应用到自己的 AI 工作流或本地项目中，请点击右侧的 `Releases` 或直接下载 `ppt-making-sop.zip` 压缩包进行解压导入。
